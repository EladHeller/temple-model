# דגם בית המקדש לפי מסכת מידות

זהו דגם לימודי פרוצדורלי ל־Blender. הסקריפט יוצר את הר הבית, עזרת הנשים,
העזרה, המזבח, הכבש והיכל סכמטי בקנה מידה עקבי.

## הפעלה מהירה

1. מתקינים Blender 4.2 ואילך.
2. פותחים את Blender ועוברים ללשונית **Scripting**.
3. פותחים את `temple_middot.py` ולוחצים **Run Script**.
4. עוברים למצלמה בעזרת מקש `0` בלוח המספרים, או מסתובבים חופשי בתצוגה.

## אפליקציית HTML

הקובץ `viewer.html` הוא אפליקציית עמוד אחד בעברית המציגה את דגם החוץ ואת
פנים ההיכל עם כלי הקודש. אפשר לעבור ביניהם, לבחור אזורים וכלים במפת התוכן,
לקרוא הסבר ומידות, לפתוח את מקור הטקסט, לסובב, להתקרב, לעבור למסך מלא
ולהוריד את קובץ ה־GLB הפעיל. הממשק מותאם גם לטלפון.
מתיקיית הפרויקט מפעילים:

```bash
python3 -m http.server 8000
```

ואז פותחים את [http://localhost:8000](http://localhost:8000). הקובץ
`index.html` מפנה אוטומטית לאפליקציה.
רכיב התצוגה נטען מה־CDN הרשמי של `<model-viewer>`, ולכן נדרש חיבור לאינטרנט
בעת פתיחת הדף. הדגם עצמו נשאר מקומי ואינו מועלה לשירות חיצוני.

## פרסום ב־GitHub Pages

הפרויקט כולל workflow מוכן ב־`.github/workflows/pages.yml`. בכל push לענף
`master` או `main` הוא אורז אתר סטטי נקי ומפרסם רק את `viewer.html`, שני
דגמי ה־GLB ושתי תמונות התצוגה. קובצי Blender וסקריפטי Python נשארים במאגר
כמקור, אך אינם נכללים באתר המפורסם.

לאחר ה־push הראשון נכנסים ב־GitHub אל **Settings → Pages** ובוחרים תחת
**Build and deployment → Source** את **GitHub Actions**. לאחר שה־workflow
מסתיים, האפליקציה זמינה בכתובת Pages של המאגר. כל הנתיבים באפליקציה יחסיים,
ולכן היא פועלת גם באתר פרויקט מהצורה `username.github.io/repository/`.

קובצי הפנים נוצרים בעזרת:

```bash
blender --background --python temple_interior.py -- \
  --save temple_middot_interior.blend \
  --export temple_middot_interior.glb \
  --render
```

אפשר גם להריץ משורת הפקודה:

```bash
blender --background --python temple_middot.py -- \
  --save temple_middot.blend \
  --export temple_middot.glb \
  --render
```

הפקודה יכולה ליצור שלושה קבצים:

- `temple_middot.blend` — קובץ העבודה המלא.
- `temple_middot.glb` — קובץ לצפייה בדפדפן או במנוע תלת־ממד.
- `temple_middot_preview.png` — תמונת תצוגה מהמצלמה המוגדרת.

לדגם הפנים יש שלושה קבצים מקבילים בשם `temple_middot_interior`: קובץ
Blender, קובץ GLB ותמונת תצוגה. דגם הפנים מציג בחתך פתוח את האולם, הקודש,
אמה טרקסין, קודש הקודשים, שתי הפרוכות, מזבח הזהב, שולחן לחם הפנים, המנורה
ואבן השתייה. שמות האוספים, האובייקטים, החומרים והתוויות נכתבו בעברית.

הגרסה המפורטת מוסיפה בין היתר את פרוכת האולם, גפן הזהב, נברשת הילני,
שרשרות ועטרות, קורות ארז ושני שולחנות; 38 תאי היכל בשלוש קומות, המסיבה,
חלונות עליונים, מעקה וכליא עורב; וכן יסוד מזבח בצורתו החלקית, סובב, מקום
הילוך, שלוש מערכות, תפוח, ספלי ניסוך, כבשים קטנים, כלי מזבח וכיור בעל
שנים־עשר דדים. חומות העזרה כוללות כעת שבעה בתי־שער מפורטים, דלתות, צירים,
מסמרים וכרכובים; עליית שומרים מעל שער הניצוץ; בית המוקד הכיפתי וארבע
לשכותיו; שתי לשכות שער ניקנור; ושש לשכות העזרה עם מתקניהן — בור הגולה
וגלגלו, ספסלי לשכת הגזית, בית הטבילה שעל גג בית הפרווה, מתקני המלח,
ההדחה והעצים.

ארבע לשכות עזרת הנשים מפורטות כחצרות לא־מקורות של 40×40 אמה, עם פתחים,
ספים, דלתות, כרכובי אבן, ריצוף וניקוז. בלשכת הנזירים נראים דודי השלמים,
מוקדי אש, שיער תחת הדוד, ספסל גילוח ותער; בלשכת דיר העצים — ערמות עצים
כשרים ופסולים, סימני תולעת ושולחן בדיקה; בלשכת המצורעים — מקווה, מעלות,
מאחז וספסל הכנה; ובלשכת בית שמניה — מדפים, קנקני חרס ליין ולשמן, כלי
מידה ושולחן מזיגה. התפקידים, המידה והיעדר הגגות מבוססים על המקורות;
צורת הפתחים, הריהוט והמתקנים היא המחשה סכמטית.

## קנה מידה וכיוונים

- ברירת המחדל היא אמה אחת = 0.5 מטר. אפשר לשנות את `CUBIT_M` בראש הקוד.
- בסצנת Blender ציר `+X` פונה מערבה, `+Y` דרומה, `-Y` צפונה ו־`+Z` למעלה.
  כך נשמרת מערכת צירים ימנית: כאשר מביטים מערבה, הצפון נמצא בצד ימין.
- כל המידות במילון `SOURCE` נכתבו באמות, פרט למקום שמצוין בו טפח.
- מידות המבוססות על פרשנות מתועדת נמצאות ב־`INTERPRETED`.
- השלמות גאומטריות שאינן מוכרעות נמצאות ב־`ASSUMED`.
- רוחב החיל מבוסס על הפרשנות המסורתית הרווחת המובאת בוויקיפדיה, ולא על
  מספר מפורש במסכת מידות.

## מה סכמטי או משוער

מסכת מידות אינה נותנת תוכנית בנייה מלאה במובן המודרני. לכן המרכיבים הבאים
מסומנים בדגם כהשלמות:

- המיקום המדויק של העזרות בתוך ריבוע הר הבית.
- עובי וגובה חלק מהחומות.
- רוחב החיל בדגם הוא 10 אמות, ולכן גבולו החיצוני 342×155 אמות; זו
  הפרשנות המסורתית הרווחת ולא לשון מספרית מפורשת במשנה.
- צורת עמודי הסורג והמרווח ביניהם.
- המזבח ממוקם מדרום לציר האמצע, כך שהכבש מדרומו וטבעות השחיטה ושאר אזור
  העבודה מצפונו; בציר מזרח–מערב נשמרים 22 אמה משער ניקנור ו־22 אמה מן האולם.
- המרווחים בין שערים שאין להם מיקום מספרי מפורש.
- מידותיהם, צורתם ומיקומם המדויק של בתי־השער, שתי לשכות ניקנור ושש לשכות
  העזרה. הגגות החתוכים בחלק מן הלשכות הם אמצעי תצוגה לימודי.
- פרטי הדלתות, הכרכובים, העמודים, מתקני הקשירה והמחסנים. שער ניקנור מובחן
  בדלתות נחושת בהתאם למסורת; שאר הדלתות מוצגות בעץ ונחושת באופן סכמטי.
- המבנה האדריכלי של הפתח המזרחי לעזרת הנשים אינו מפורט במקור ולכן הוא
  מסומן בנפרד כרכיב סכמטי; הוא אינו שער ניקנור, הנמצא ממערב לו בראש המעלות.
- צורת כיפת בית המוקד, חלוקת ארבע לשכותיו ומתקני האש שבתוכו.
- המידות והחלוקה הפנימית של 38 תאי ההיכל (מספרם ופיזורם הכללי ממודלים,
  אך רוחב כל תא ופתחיו סכמטיים).
- פרטי עיטור, חומרי גמר וצבעים.

אלמנטים פרשניים וסכמטיים קיבלו חומר כחול־אפור ונמצאים באוספים ששמם כולל
"פרשנות" או "סכמטי", כך שאפשר לזהותם ולהסתירם בקלות.

## מבנה הקובץ

המודל מחולק לאוספים ממוספרים ב־Outliner של Blender: הר הבית, חומות,
הסורג, עזרת נשים, העזרה, המזבח, ההיכל, תאים, תאורה ותוויות. המבנה הזה מאפשר
לעבוד בהדרגה בלי לפרק את הדגם כולו.
