# דגמי המשכן ובית המקדש

מאגר זה כולל שני שחזורים לימודיים פרוצדורליים ל־Blender, המופרדים לחלוטין
בתיקיות:

- `tabernacle/` — המשכן במדבר, על בסיס ספר שמות ושיטת חז״ל.
- `temple/` — בית המקדש השני, על בסיס מסכת מידות ומקורות נלווים.

בכל תיקייה נמצאים קוד המקור, קובצי Blender, דגמי GLB, תמונות תצוגה ואפליקציית
HTML עצמאית. הקובץ `index.html` בשורש משמש דף כניסה לבחירת הדגם.

## הפעלה מהירה

מתיקיית המאגר מפעילים שרת מקומי:

```bash
python3 -m http.server 8000
```

לאחר מכן פותחים את [http://localhost:8000](http://localhost:8000). רכיב
`<model-viewer>` נטען מה־CDN הרשמי ולכן נדרש חיבור לאינטרנט; קובצי התלת־ממד
עצמם נשארים מקומיים.

## המשכן במדבר

הדגם הראשי מציג חצר של 100×50 אמה, קלעים ועמודים, שער מזרחי, מזבח נחושת
5×5×3 אמות, כיור ומבנה משכן מסורתי בגודל 30×10×10 אמות. המשכן מוצג בכוונה
ללא יריעות הגג והמכסים, וקרשי הדרום מונמכים כחתך, כדי לחשוף את מסך הפתח,
הקודש, הפרוכת, קודש הקודשים, ארון העדות והכרובים, שולחן לחם הפנים, המנורה
ומזבח הקטורת. דגם הפנים הנפרד מציג את אותו חתך בתקריב.

מזבח העולה תוקן לפי פרק „במשכן” בערך
[מזבח העולה](https://he.wikipedia.org/wiki/מזבח_העולה): גוף עצי שיטים מצופה
נחושת וחלול שממולא באדמה בעת החניה, רשת נחושת עד חצי הגובה, יסוד בולט,
קרנות שהן חלק מגוף המזבח, טבעות ובדים, מערכת עצים ואש תמיד. ממדי הכבש ורוחב
בליטת היסוד אינם נתונים בערך ולכן הם מוצגים כחלקים סכמטיים כחולים־אפורים.

שולחן הפנים תוקן לפי הערך
[שולחן הפנים](https://he.wikipedia.org/wiki/שולחן_הפנים): נוספו מסגרת בגובה
טפח, שני זרי זהב, ארבע טבעות ובדים, ארבעה סניפים, 28 קני זהב, שתי מערכות של
שישה לחמים בצורת „תיבה פרוצה” ושני בזיכי לבונה. במחלוקות המתוארות בערך
נבחרו מסגרת מעל הטבלה, סניפים ישרים ובזיכים בתוך הלחם העליון; הסניפים
מסומנים כחלק פרשני.

ליצירה מחדש של שני הדגמים, מתוך `tabernacle/`:

```bash
blender --background --python tabernacle.py -- \
  --save tabernacle.blend --export tabernacle.glb --render

blender --background --python tabernacle.py -- \
  --interior --save tabernacle_interior.blend \
  --export tabernacle_interior.glb --render
```

### בסיס מקורות המשכן

- שמות כה — הארון, הכפורת והכרובים, השולחן והמנורה.
- שמות כו — היריעות, המכסים, הקרשים, האדנים, הבריחים, הפרוכת ומסך הפתח.
- שמות כז — מזבח העולה וחצר המשכן.
- שמות ל — מזבח הקטורת והכיור.
- שמות לה–מ — תיאור הביצוע והקמת המשכן.
- [המשכן — ויקיפדיה העברית](https://he.wikipedia.org/wiki/המשכן), ובפרט
  פרקי מבנה המשכן ושיטת חז״ל והפרשנות המסורתית.

בהתאם לשיטת חז״ל המוצגת בערך, הדגם בוחר בחלוקה של 20 אמה לקודש ו־10 אמות
לקודש הקודשים, מציב את המשכן 20 אמה מקלעי הצפון, הדרום והמערב ו־50 אמה מן
המזרח, וממקם את מזבח הקטורת, השולחן והמנורה לפי המרחקים היחסיים המסורתיים.
לקרשים נבחר עובי קבוע של אמה כדעת רבי נחמיה; דעת רבי יהודה, שלפיה הקרשים
הולכים וצרים כלפי מעלה, אינה מוצגת בגרסה זו.

התורה אינה מספקת תוכנית הנדסית מלאה. מידות הכיור, המכסים העליונים, צורת
הכרובים, עובי האריגים, פרטי הפינות והחיבורים וחלק מצורת הכלים מסומנים בקוד
ב־`ASSUMED` או בחומר כחול־אפור. הפרוכת ומסך פתח האוהל מוצגים כיריעות בד
שלמות וישרות בעלות מרקם ארוג ורקמה מוגבהת בתכלת, ארגמן, תולעת שני ושש.
לרקע נבחר גוון כחול־ארגמני כהה כהמחשה אמנותית בלבד; המקורות אינם קובעים
כך את מראה הרקע המוגמר. בפרוכת נוספו דמויות כרובים רקומות, ובמסך הפתח דגם
מעוינים רקום.

## בית המקדש

דגם המקדש נמצא בתוך `temple/`. הוא כולל דגם חוץ של הר הבית והעזרות ודגם
חתך של האולם, ההיכל וכלי הקודש. ליצירה מחדש, מתוך
`temple/`:

```bash
blender --background --python temple_middot.py -- \
  --save temple_middot.blend --export temple_middot.glb --render

blender --background --python temple_interior.py -- \
  --save temple_middot_interior.blend \
  --export temple_middot_interior.glb --render
```

המידות המפורשות של המקדש נמצאות במילון `SOURCE`, פרשנויות מתועדות נמצאות
ב־`INTERPRETED`, והשלמות גאומטריות נמצאות ב־`ASSUMED`.

בדגם החוץ לשכת הגזית מוצגת בפינה הצפון־מזרחית של העזרה לפי שיטת הרמב״ם
המובאת בוויקיפדיה: מחציתה הדרומית בקודש ומחציתה הצפונית בחיל. יש בה שני
פתחים ומושבי סנהדרין בחצי גורן בצד המערבי. שיטת תוספות, הממקמת אותה
בצפון־מערב, אינה מוצגת. מידות המבנה וסידור הריהוט סכמטיים.

## קנה מידה וכיוונים

- בשני הדגמים אמה אחת = 0.5 מטר.
- במשכן ציר `+X` פונה מערבה, `+Y` דרומה ו־`+Z` למעלה.
- במקדש קלט התוכנית משתמש ב־`+Y` לצפון ומומר לסצנת Blender ימנית; פרטי
  המימוש מתועדים בראש `temple/temple_middot.py`.
